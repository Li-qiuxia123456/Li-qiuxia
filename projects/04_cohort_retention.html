用户留存率与 Cohort 分析

按用户首月划分 Cohort，计算第 N 周/月的留存率矩阵，直观呈现产品粘性和用户生命周期价值。

📖核心知识点

👥 Cohort 基本概念
按用户"注册月/首次活跃月"分组，纵向追踪同一批用户在后续月份的留存。行=Cohort，列=第 N 月留存。
📅 月差计算
两个月份字符串转 pd.Period，相减得到 month_offset（N 个月后）。
🔄 pivot_table 透视
pd.pivot_table：index=Cohort，columns=month_offset，values=user_id 去重计数。
🎨 可视化矩阵表格
百分数表格（热力图效果），容易识别留存"断崖"位置。可用 style.background_gradient 进一步美化。
🧹 数据去重与去噪
同一用户同月可能有多次记录，需 drop_duplicates 去重，避免被重复计入。
📈 用户规模与留存解读
大 cohort 更稳定，小 cohort 波动大。解读需结合样本量与业务事件（如版本发布、活动等）。

💻核心函数速查

# 基础聚合
df.groupby('active_month')['user_id'].nunique()

# 去重
df = df.drop_duplicates(['user_id', 'active_month'])

# Period 转换
df['first_period'] = pd.PeriodIndex(df['first_active_month'], freq='M')
df['active_period'] = pd.PeriodIndex(df['active_month'], freq='M')
df['month_offset'] = (df['active_period'] - df['first_period']
                      ).apply(lambda x: x.n)

# pivot_table 矩阵
cohort_counts = df.pivot_table(
    index='first_active_month',
    columns='month_offset',
    values='user_id',
    aggfunc='nunique'
)

# 归一化（以初始 cohort 数量为基准）
cohort_size = cohort_counts[0]
retention = cohort_counts.divide(cohort_size, axis=0)
retention = retention.round(3) * 100

# 格式化
retention_style = retention.style.format('{:.1f}%')

🎯分析目标

本项目将围绕 Cohort 留存，按以下步骤展开分析：

  • 去重处理：按 user_id + active_month 去重，避免重复计入
  • 基础统计：按月份统计活跃用户数、各 cohort 初始用户数
  • month_offset 计算：活跃月 - 首月 = 第 N 月
  • pivot_table 矩阵：行=Cohort，列=month_offset
  • 留存率归一化：除以 cohort 初始用户数得到百分比
  • 打印热力表：各 cohort 在各月的留存率
  • 首月 cohort 留存对比：对比不同 cohort 的 M1/M2 留存差异

📋数据集字段说明

user_id用户唯一标识
first_active_month用户首次活跃月份（如 2024-01）
active_month该用户该月是否活跃
