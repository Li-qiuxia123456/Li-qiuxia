RFM 用户分层

基于消费 Recency、Frequency、Monetary 三个维度做用户价值分层，识别重要价值用户和流失预警用户。

📖核心知识点

RFM 模型原理
R(Recency 最近购买距今越近越好)、F(Frequency 购买频率越高越好)、M(Monetary 消费金额越高越好)。经典用户分群方法，将用户划分为不同价值层级。
🧮 分位数计算与打分
pd.Series.quantile([0.2, 0.4, 0.6, 0.8]) 计算分位数。R 越小分数越高（反转），F/M 越大分数越高。
📊 groupby 多级聚合
df.groupby('user_id').agg({'order_date': lambda x: (today-x.max()).days, 'order_date': 'count', 'amount': 'sum'}) 可以对同一分组应用多种聚合。
🏷️ 用户分层标签
典型分层：重要价值用户(R 高,F 高,M 高)、重要保持用户、重要发展用户、一般用户、流失预警用户。通过条件判断或规则组合生成分层标签。
⏰ Pandas 时间运算
pd.to_datetime()、时间差计算、日期归一化等。dt.days 可以提取时间差的天数。
📈 统计占比与占比累计
各层用户数/总用户数、各层总消费/总金额，帮助识别头部用户价值。典型结论是少数用户贡献了大部分收入（帕累托法则）。

💻核心函数速查

# 日期处理
df['order_date'] = pd.to_datetime(df['order_date'])
today = df['order_date'].max()

# RFM 聚合
rfm = df.groupby('user_id').agg(
  R=('order_date', lambda x: (today - x.max()).days),
  F=('order_date', 'count'),
  M=('amount', 'sum')
).reset_index()

# 分位数打分
rfm['R_score'] = pd.qcut(rfm['R'], 5, labels=[5,4,3,2,1]).astype(int)
rfm['F_score'] = pd.qcut(rfm['F'].rank(method='first'), 5, labels=[1,2,3,4,5]).astype(int)
rfm['M_score'] = pd.qcut(rfm['M'].rank(method='first'), 5, labels=[1,2,3,4,5]).astype(int)

# 条件分层
rfm.loc[条件, 'segment'] = '重要价值用户'

# 各层占比统计
rfm.groupby('segment')['M'].sum().sort_values(ascending=False)
