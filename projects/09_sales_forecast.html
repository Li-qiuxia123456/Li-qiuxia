销售预测与季节性分析

基于历史销售时间序列，检测季节性、趋势性，并使用移动平均与简单回归预测未来销售额。

🎉 恭喜！你已完成本项目。继续挑战下一个吧 →

📖核心知识点

时间序列格式
PeriodIndex / DatetimeIndex，结合 rolling 窗口分析。
📈 移动平均 SMA
df.rolling(window=3).mean() 平滑噪声，识别趋势。
📊 同比 YoY
(本月 - 去年同月) / 去年同月，衡量增长幅度。
🔄 环比 MoM
(本月 - 上月) / 上月，衡量短期波动。
📉 线性回归趋势
sklearn LinearRegression，以时间索引作为 X，预测销售额。
🎄 季节性识别
按月均值分析（11/12 月明显高，夏季也高），输出季节因子。

💻核心函数速查

# 日期解析
df['date'] = pd.to_datetime(df['date'])
df = df.set_index('date').sort_index()

# 移动平均
df['ma3'] = df['sales'].rolling(3).mean()
df['ma6'] = df['sales'].rolling(6).mean()
df['ma12'] = df['sales'].rolling(12).mean()

# 同比/环比
df['yoy'] = df['sales'].pct_change(12) * 100
df['mom'] = df['sales'].pct_change(1) * 100

# 线性回归
from sklearn.linear_model import LinearRegression
df['month_idx'] = range(len(df))
X = df[['month_idx']]
y = df['sales']
model = LinearRegression().fit(X, y)
df['trend'] = model.predict(X)

# 季节因子（同月均值 / 总均值）
seasonal = df.groupby(df.index.month)['sales'].mean()
season_factor = seasonal / seasonal.mean()

🎯分析目标

本项目将围绕销售预测，按以下步骤展开：

  • 日期处理：解析日期字段并排序
  • 3/6/12 月 MA：平滑噪声识别趋势
