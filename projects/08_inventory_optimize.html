库存优化与 ABC 分析

ABC 分析法（帕累托法则）将 SKU 按价值贡献分成三类，A 类重点管理、C 类简化流程，同时计算周转率与缺货风险。

📖核心知识点

ABC 分析
按年度销售金额从高到低排序，累计 0-70% 为 A 类，70-90% 为 B 类，90-100% 为 C 类。
🧮 销售额计算
annual_revenue = avg_daily_sales × 365 × unit_cost。
📊 累计占比
cumsum() / sum() 用于累计贡献率曲线。
📦 库存天数
stock / avg_daily_sales 可覆盖多少天销售。
📈 分类统计
groupby('abc_class') 看各类 sku 数量、金额占比、平均库存天数。
⚠️ 缺货风险识别
库存天数 < 安全阈值的商品优先补货。

💻核心函数速查

# 年销售额计算
df['annual_revenue'] = df['avg_daily_sales'] * 365 * df['unit_cost']

# 按年销售额降序
df = df.sort_values('annual_revenue', ascending=False).reset_index(drop=True)

# 累计占比
df['cum_revenue'] = df['annual_revenue'].cumsum()
df['cum_ratio'] = df['cum_revenue'] / df['annual_revenue'].sum()

# ABC 分类
def abc_class(ratio):
    if ratio <= 0.70: return 'A'
    elif ratio <= 0.90: return 'B'
    else: return 'C'
df['abc_class'] = df['cum_ratio'].apply(abc_class)

# 库存天数
df['stock_days'] = df['stock'] / df['avg_daily_sales']

🎯分析目标

本项目将围绕库存优化与 ABC 分析，按以下步骤展开：

  • 年销售额计算：avg_daily_sales × 365 × unit_cost
  • 排序 + 累计占比：按年销售降序，计算累计贡献
  • ABC 分类标签：0-70% A 类、70-90% B 类、90-100% C 类
  • 各类占比表：每类 sku 数、金额占比、平均库存天数
  • 库存天数 + 缺货风险 Top：找出需优先补货的商品
  • 建议补货：对高销售 + 低库存的 sku 提出补货建议
