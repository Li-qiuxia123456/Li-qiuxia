用户流失预测建模

用 Pandas 提取用户行为特征（活跃度、消费频次、最近活动时间等），配合分类标签训练 sklearn 模型，预测高流失风险用户。

📖核心知识点

🧩 特征工程基础
从原始行为数据中提取有意义的数值字段。活跃天数、订单数、金额、会话时长等，这些是模型判断用户是否流失的"线索"。
✂️ 训练集/测试集切分
sklearn.model_selection.train_test_split 划分训练/测试集，常用参数 test_size、random_state（保证复现）。
🌲 分类算法
逻辑回归、随机森林、XGBoost 等。建议从简单模型（LogisticRegression）起步，逐步尝试更复杂的 ensemble 模型比较效果。
📊 分类评估指标
accuracy、precision、recall、F1、ROC-AUC。imbalanced 数据尤其要关注 recall / precision，单一 accuracy 会误导。
⭐ 特征重要性
tree 类模型可以打印 feature_importances_，理解模型判断用户流失的依据，也便于向业务方解释。
🔢 Pandas 数值统计与描述
df.describe()、df.corr() 相关矩阵快速发现高相关字段，也可检查是否存在多重共线性。

💻核心函数速查

# 数据概览
df.head()
df.describe()
df.corr()                # 相关矩阵
df['is_churned'].value_counts()

# 特征选择
feature_cols = ['days_active', 'total_orders', 'total_amount',
                'avg_session_min', 'last_active_days_ago']
X = df[feature_cols]
y = df['is_churned']

# 切分与训练
from sklearn.model_selection import train_test_split
X_train, X_test, y_train, y_test = train_test_split(
    X, y, test_size=0.3, random_state=42)

from sklearn.linear_model import LogisticRegression
from sklearn.ensemble import RandomForestClassifier

lr = LogisticRegression(max_iter=1000).fit(X_train, y_train)
rf = RandomForestClassifier(n_estimators=100, random_state=42).fit(X_train, y_train)

# 评估
from sklearn.metrics import (accuracy_score, confusion_matrix,
                             classification_report)
y_pred = rf.predict(X_test)
print(accuracy_score(y_test, y_pred))
print(confusion_matrix(y_test, y_pred))

# 特征重要性
pd.DataFrame({'feature': feature_cols,
              'importance': rf.feature_importances_}
            ).sort_values('importance', ascending=False)
