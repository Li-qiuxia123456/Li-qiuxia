营销活动 ROI 评估

结合活动投入成本与实际转化收益，计算各渠道/活动的投资回报率，识别高 ROI 与低效渠道。

📖核心知识点

📐 基础指标定义
ROI = (revenue - spend) / spend；CPC = spend / clicks；CPA = spend / conversions；CTR = clicks / impressions。
➕ 新增计算列
df['roi'] = (df['revenue'] - df['spend']) / df['spend']，可以一次生成多个衍生列。
🔎 按渠道聚合
df.groupby('channel') 评估各渠道整体表现，判断渠道质量与预算分配。
📦 活动维度聚合
df.groupby('campaign') 看哪些活动效果最好，作为后续活动参考。
⬇ 排序与筛选
df.sort_values('roi', ascending=False)，筛选 ROI 高/低的活动做重点分析。
💡 业务解读
高 ROI 但规模小的渠道要不要加预算？低 ROI 大渠道要不要砍？必须结合业务目标综合评估。

💻核心函数速查

# 新增 ROI / CPC / CPA 等列
df['roi'] = (df['revenue'] - df['spend']) / df['spend']
df['cpc'] = df['spend'] / df['clicks']
df['cpa'] = df['spend'] / df['conversions']
df['profit'] = df['revenue'] - df['spend']

# 按渠道聚合
by_channel = df.groupby('channel').agg({
    'spend': 'sum',
    'revenue': 'sum',
    'clicks': 'sum',
    'conversions': 'sum'
}).reset_index()
by_channel['roi'] = (by_channel['revenue'] - by_channel['spend']) / by_channel['spend']

# 排序
df.sort_values('roi', ascending=False).head(5)
df.sort_values('profit', ascending=False).tail(5)

# 筛选
high_roi = df[df['roi'] > 1.0]
low_roi = df[df['roi'] < 0.3]
