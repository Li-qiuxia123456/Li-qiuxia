客户终身价值 LTV 预测

基于历史消费数据预测用户的终身价值（LTV），帮助资源倾斜到高价值用户上。

📖核心知识点

LTV 基本公式
LTV = 历史累计消费 / (1 - 流失率) × 留存月数，或简化为 avg_monthly × 预测生命周期月数。
🧮 历史消费聚合
total_amount 求和，avg_monthly = total / months。
📊 用户年限分布
months 的分布决定了是否"成熟用户"，区分长期与新用户。
📈 线性回归/中位数预测
简单但稳健的起点，用历史月均 × 预测生命周期月数。
🎯 用户分层
高 LTV / 中 LTV / 低 LTV，结合首 3 月消费做早期 LTV 预测。
💡 业务应用
高 LTV 用户给予 VIP 服务、专属优惠券，提升留存。

💻核心函数速查

# 字符串切分 monthly_amount（注意引号）
df['monthly_list'] = df['monthly_amount'].str.strip('"').str.split(',')
df['monthly_list'] = df['monthly_list'].apply(lambda x: [int(i) for i in x])

# 总消费与月均
df['total_amount'] = df['monthly_list'].apply(sum)
df['avg_monthly'] = df['total_amount'] / df['total_months']

# 基于中位数生命周期预测
median_months = df['total_months'].median()
df['ltv_median'] = df['avg_monthly'] * median_months

# 用户分层
df['segment'] = pd.qcut(df['ltv_median'], q=3, labels=['低价值', '中价值', '高价值'])

🎯分析目标

本项目将围绕客户终身价值预测，按以下步骤展开分析：

  • 历史汇总：计算每位用户的累计消费、月均消费
  • 用户年限分布：看看 total_months 分布
  • 预测生命周期：基于中位数/分位数预测未来月数
  • LTV = 月均 × 生命周期：输出每个用户的预测终身价值
  • 用户分层：按 LTV 分档，高/中/低价值
  • Top 高价值用户名单：找出需重点服务的用户
