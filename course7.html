库存销量预测分析

使用历史销量数据与移动平均法，对未来销量做预测，指导补货与库存管理。

🎯 学习目标
1 理解移动平均预测 / 2 手动计算 N 期移动平均 / 3 输出预测值并用于补货建议
👥 适合人群
供应链/库存管理、采购计划、零售运营。
📖 前置知识
Python 基础语法。
时间序列移动平均库存预测供应链
📖

课程核心知识点

点击展开每个知识点，了解详细原理与实际应用

1 移动平均基础▶

用最近 N 期销量的均值作为下一期预测，平滑短期波动。

2 N 期移动平均计算▶

取最近 N 个时间窗口，依次求均值得到预测序列。

3 补货决策建议▶

预测值 + 安全库存共同决定补货量，避免缺货与积压并存。
